# Quantora Analytics - Premium Research Platform

Quantora Analytics is a world-class, futuristic financial research and analytics platform. This website is designed with an institutional-grade aesthetic, combining the authority of firms like McKinsey and Goldman Sachs with the technological edge of modern AI startups.

## Key Features
- **Cinematic Hero Section**: High-resolution visuals with floating analytics dashboards.
- **Institutional Design System**: Deep navy and midnight black palette with electric blue accents and glassmorphism.
- **Advanced Animation System**: Scroll-triggered reveals, 3D tilt effects on cards, and a custom interactive cursor.
- **Market Intelligence Hub**: Real-time ticker, categorised research reports, and AI-driven insights.
- **Specialized Research**: Dedicated sections for Equity, Macro, and Political research.
- **Futuristic R&D Lab**: Visual representation of neural network architectures and innovation roadmaps.

## Technology Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Grid, Glassmorphism, Backdrop Filters)
- **Interactions**: Vanilla JavaScript (Intersection Observer, 3D Transform API)
- **Icons**: Lucide (via CDN)
- **Typography**: Inter (via Google Fonts)

## Performance
- Optimized for smooth scrolling and GPU-accelerated animations.
- Fully responsive across all devices (Desktop, Tablet, Mobile).
- Light-weight and high-performance.
